# Windows 小工具集

这个目录包含了一些实用的 Windows 批处理脚本工具，主要用于简化常见的系统管理任务。

## 可用工具

### 🔄 pkill - 端口进程管理工具
**功能**: 通过端口号快速查找并终止占用该端口的进程

**主要特性**:
- 智能识别单/多进程场景
- 交互式进程选择确认
- 支持 taskkill 命令的所有参数
- 显示完整的网络连接信息

**详细文档**: [pkill/README.md](pkill/README.md) | [pkill/README_CN.md](pkill/README_CN.md) (中文版)

**使用示例**:
```bat
pkill 8080          # 终止占用 8080 端口的进程
pkill 3000 /t       # 终止进程及其所有子进程
pkill 5432 /v       # 终止进程并显示详细信息
```

## 安装说明

### 添加到系统 PATH（推荐）
1. 选择一个固定位置存放工具脚本
2. 将脚本所在目录添加到系统的 PATH 环境变量
3. 重启命令提示符窗口即可使用

### 使用方法
- 直接运行脚本文件
- 或通过命令提示符调用
- 部分工具需要管理员权限（通常会有提示）

## 注意事项
- 使用前请确认工具的用途，避免误操作
- 某些操作可能影响系统稳定性，请谨慎使用
- 建议在测试环境先行验证工具功能
